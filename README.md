## my-twitter

JavaScript Full Stack Project

## For dev

in the twitter folder

```bash
# run this bash first time or when new packge is added
npm i

# open front end dev server
yarn start
```

in the twitter_api folder

```bash
# run this bash first time or when new packge is added
npm i

# open back end node server dev mode
npm run dev
```
